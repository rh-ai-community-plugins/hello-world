# Documentation

This directory contains all project documentation, organized by topic.

## Sections

### [Architecture](architecture/)

How the RHOAI Dashboard plugin system works, including the extension contract, Module Federation config, and deployment models. Also covers example community plugins and key reference files in the dashboard codebase.

### [Development](development/)

Local development environment setup, dashboard integration workflow, and reference for the dashboard backend APIs available to plugins.

### [Deployment](deployment/)

Deploying the plugin on OpenShift with Helm, registering it with the RHOAI Dashboard, and configuring the BFF proxy.

### [Archives](archives/)

Historical planning documents kept for reference.
